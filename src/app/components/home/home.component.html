Simple CSS application

Welcome to my CSS angular application!

This application is made to help developer use some common CSS tools in one place. I made this application to improve my Angular and CSS skills, and also to help myself writting CSS files in the future. The tools on your left are quite simple, but I think you will find it fairly useful. I certainly hope that this tool will help you save some time, and will make you feel this time was not wasted.

The tools description

At the moment there are only three tools, but I'm planning to add new ones in the future. Here is current tools description:

  • Shape generator - this tool helps to make some more unusual shape from div than simple square or rectangle.
  • Gradient generator - as the name suggests, it makes a gradient of chosen colors, and rotation degree.
  • Banner generator - have you ever seen a simple post on facebook, with a few words and some color or gradient in the background? This is exactly what this tool is made for. It also lets you download your effects of work.
